Detailed Description

syslog publisher_api implementation. This plugin sends messages to syslog synchronously. Multiple independent instances of this plugin may be used simultaneously, but the underlying syslog setting (priority) will be that of the most recently created instance.

If not configured, the default priority is LOG_INFO. env("ADC_SYSLOG_PLUGIN_PRIORITY") overrides the default if defined with a priority string from /usr/include/sys/syslog.h.
